# HR-app 

## How to run the app

For React:
```
cd HR-app/React/hr-app/
npm install

//to starte the frontend
npm start
```

For Django:
```
// install dependencies
cd HR-app/
python -m venv env
env/Scripts/activate
pip install django
pip install django-cors-headers

//to start the backend
python manage.py runserver
```
